# Netflix-Website-Squid-Game

Netflix Website "Squid Game" landing page built using HTML, CSS and JavaScript — a small fan-made recreation of the Netflix show landing experience for learning and demo purposes.

![Landing screenshot](./Screenshot%202024-06-19%20005917.png)

## Demo
Open `index.html` in your browser to view the page locally.

## Features
- Responsive single-page layout built with HTML and CSS
- Simple JavaScript interactions
- Includes assets and styles to mimic the Netflix landing page look-and-feel

## Files
- `index.html` — main page
- `style.css` — styles
- `script.js` — small JS interactions
- `items/` — folder containing image assets used in the page
- `Screenshot 2024-06-19 005917.png` — included screenshot shown above

## How to run
1. Clone the repo:

   git clone https://github.com/BinaryVortex/Netflix-Website-Squid-Game.git
2. Change into the project directory:

   cd Netflix-Website-Squid-Game
3. Open `index.html` in your browser (double-click or use a local server):

   - Double-click `index.html` or
   - Serve with a simple server (Python):

     python3 -m http.server 8000
     # then open http://localhost:8000
